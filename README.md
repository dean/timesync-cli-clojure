# Timesync CLI

A basic CLI for the TimeSync[http://github.com/osuosl/timesync] application.

## Installation

Get the repo and install lein.

```sh
git clone https://github.com/dean/timesync-cli-closure
```

Mac:
```sh
brew install lein
```

Ubuntu:
```sh
sudo apt-get install lein
```
